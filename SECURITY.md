# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

You can Report vulnerabilities through  DM'ing dadarkcube on Discord. Responses to vulnerability reports usually take
around 3 hours to more than a day, depending on how difficult fixing a
vulnerability is.
